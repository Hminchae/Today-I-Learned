iOS 13을 기점으로 달라진 비동기 동기


| Sync         | Async              |
| ------------ | ------------------ |
| serial(main) | concurrent(global) |


```swift
override func viewDidLoad() {
        super.viewDidLoad()

        view.backgroundColor = .white
        print("1")
        
        DispatchQueue.main.async {
            print("2")
        }
        
        print("3")
    }

// 3->2->1
```
