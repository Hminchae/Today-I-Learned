##### RestAPI <- 18회차 강의자료
네트워크를 통해서 핵심 콘텐츠와 기능을 활용할 수 있도록 제공되는 인터페이스, 아키텍쳐 스타일

##### 그래서 단점?
- Overfetching
	- 필요한 정보값보다 더 많은 정보값이 로딩될 수 있음
	- ex)네이버 영화 API에서 
- Underfetching 
	- 필요한 정보보다 부족한 정보 로딩으로 인해 추가 API 요청이 필요
- Endpoint
	- 서비스 규모가 커질수록 엔드포인트가 늘어나
